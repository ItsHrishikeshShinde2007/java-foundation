# Java Concepts

This directory contains individual Java programs focused on learning and practicing one programming concept at a time.

Each file is intentionally small and isolates a specific topic to build a strong understanding before combining multiple concepts into larger projects.

## Topics

- Variables & Data Types
- Operators
- Scanner Input
- Conditional Statements
- Loops
- Methods
- Arrays
- Classes & Objects
- Constructors
- Inheritance

These programs are intended for concept mastery rather than building complete applications.
