# Java Foundations

Collection of beginner Java programs and foundational syntax practice.

This repository contains basic Java programs written while learning core programming concepts and preparing for Semester 3 Object-Oriented Programming (OOP).

## Topics Covered

- Java syntax
- Variables and data types
- Operators
- User input using Scanner class
- Console output

## Files Included

### Variables.java
Practice program demonstrating:
- variable declaration
- primitive data types
- strings
- boolean values
- output statements

### Operators.java
Practice program demonstrating:
- arithmetic operators
- addition
- subtraction
- multiplication
- division
- modulo operator

### ScannerInput.java
Practice program demonstrating:
- Scanner class
- taking user input
- string input handling
- console interaction

## Language Used

- Java

## Purpose

Learning Java fundamentals for:
- programming foundations
- OOP preparation
- backend development basics
- problem-solving skills
